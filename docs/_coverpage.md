# blackdot

> Modular dotfiles framework for the AI-assisted development era

```bash
# macOS / Linux / WSL
curl -fsSL https://raw.githubusercontent.com/blackwell-systems/blackdot/main/install.sh | bash

# Windows (PowerShell)
irm https://raw.githubusercontent.com/blackwell-systems/blackdot/main/install-windows.ps1 | iex
```

- **Claude Code Integration** - Portable sessions, dotclaude profiles, git safety hooks
- **Deep Developer Tool Integrations** - AWS, CDK, Rust, Go, Python, NVM, SDKMAN with 90+ aliases
- **Multi-Vault Secrets** - Bitwarden, 1Password, or pass with unified API
- **Feature Registry** - Modular control plane with presets
- **Configuration Layers** - Multi-machine settings that just work
- **Self-Healing** - Health checks with auto-fix
- **Cross-Platform** - macOS • Linux • Windows • WSL2 • Docker
